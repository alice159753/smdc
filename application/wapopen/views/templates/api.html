APP接口文档

API接口注意事项：

所有接口公共参数（如下，接口里面将不再另行列出公共参数说明）：

参数名称 是否必需 参数说明
api_key 是 找技术人员分配 （不同设备会有所不同，需要技术人员分配）
req_time 是 请求接口时间（Unix时间戳+4位的随机数）
如：时间戳=1466650143；4位随机数=1234；
那么：req_time=14666501431234
api_token 是 参数说明 api_token=a84df9733d88f81433ddf1d35f2aa975
1.获取所有的待传参数，去掉键为api_key,api_secret的参数 ，加入api_token参数；
2.用函数 ksort($para);reset($para);排序参数
3.所有参数排序后以键=值的形式拼接成字符串，每个参数以&连接(最后一个参数后面去掉&)(例子 a=1&b=2&c=3).
4.md5加密

注意：所有接口code=0表示成功；文档中的app_key仅用于测试，可能会影响程序功能，不能在开发中使用
